## Intro

This is a terminal program for a 3D Minesweeper game.

## How to Compile & Run on Linux

For GCC compilation: ``g++ main.cpp display.cpp -o 3dMinesweeper``

To run: ``./3dMinesweeper``

To run and compile you can do: ``clear && g++ main.cpp display.cpp -o 3dMinesweeper && ./3dMinesweeper``
